1. Type Of System: 
Description: Type of spacing between the phase conductors.
Options: Symmetrical spacing, Unsymmetrical spacing

2. Spacing between the conductors:
Description: Spacing between the phase conductors (in m, mm, ft or inch).
Valid Range: All positive distances

3. Number of subconductors per bundle:
Description: Number of subconductors in each phase conductor (Assumed to be equal for all the three phases).
Valid Range: Positive number of subconductors
